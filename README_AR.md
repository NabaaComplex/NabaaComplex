# موقع مجمع نبأ الطبي

موقع عربي متجاوب ومناسب للفتح من رمز QR. يعرض تعريف المجمع، العيادات والخدمات الصحية، مختبر التحليلات الطبية، صورًا حقيقية من مرافق المجمع، أرقام الحجز، الموقع والحسابات الرسمية ضمن صفحة واحدة.

## الملفات

- `index.html` — هيكل الصفحة والنصوص التعريفية والرسالة والرؤية وأقسام الصور والموقع.
- `styles.css` — التصميم والألوان والاستجابة للجوال والعرض البانورامي للصور.
- `content.js` — معلومات العيادات والخدمات والمختبر ومعرض الصور وأرقام الاتصال والروابط.
- `app.js` — عرض بطاقات الخدمات والصور والبيانات والوظائف التفاعلية.
- `assets/` — شعار المجمع والأيقونة.
- `assets/complex/` — الصور الحقيقية المحسنة للاستخدام على الويب.

## تعديل المعلومات لاحقًا

افتح ملف `content.js` لتعديل:

- أسماء العيادات والخدمات ووصفها.
- تفاصيل الخدمات التي يقدمها كل قسم.
- مختبر التحليلات الطبية وفحوصاته.
- صور المعرض وعناوينها.
- أرقام الحجز.
- العنوان.
- روابط Facebook وInstagram وTikTok وGoogle Maps وWaze.

يمكن تعديل النصوص العامة مثل المقدمة و«لماذا تختارنا» والرسالة والرؤية من ملف `index.html`.

بعد التعديل، ارفع النسخة الجديدة إلى نفس الاستضافة ليبقى رابط الموقع ورمز QR كما هما.

## استضافة مجانية مقترحة

### GitHub Pages
1. أنشئ مستودعًا جديدًا.
2. ارفع محتويات هذا المجلد، وليس ملف ZIP نفسه.
3. من Settings → Pages اختر الفرع `main`.
4. استخدم الرابط الناتج في رمز QR.

### Cloudflare Pages
1. اربط مستودع GitHub.
2. لا تستخدم أمر بناء.
3. اجعل مجلد النشر هو جذر المشروع.
4. كل تحديث للمستودع يُنشر تلقائيًا.

## قاعدة مهمة لرمز QR

اجعل رمز QR يشير إلى رابط الموقع الثابت فقط. عند تحديث محتوى الموقع سيبقى رمز QR صالحًا.
