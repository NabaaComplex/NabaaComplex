# موقع مجمع نبأ الطبي

موقع عربي متجاوب، مناسب للفتح من رمز QR ويجمع معلومات المجمع في صفحة واحدة.

## الملفات

- `index.html` — هيكل الصفحة.
- `styles.css` — التصميم والألوان والاستجابة للجوال.
- `content.js` — **الملف الرئيسي لتحديث المعلومات**.
- `app.js` — عرض البيانات والوظائف التفاعلية.
- `assets/` — الشعار والأيقونة.

## تعديل المعلومات لاحقًا

افتح ملف `content.js` فقط. يمكن تعديل:

- أرقام الحجز.
- أسماء الأطباء واختصاصاتهم.
- أيام الحضور والملاحظات.
- العنوان.
- روابط Facebook وInstagram وTikTok وGoogle Maps.

ثم ارفع النسخة الجديدة إلى نفس الاستضافة. سيبقى رابط الموقع ورمز QR كما هما.

## استضافة مجانية مقترحة

### GitHub Pages
1. أنشئ مستودعًا جديدًا.
2. ارفع محتويات هذا المجلد، وليس ملف ZIP نفسه.
3. من Settings → Pages اختر الفرع `main`.
4. استخدم الرابط الناتج في رمز QR.

### Cloudflare Pages
1. اربط مستودع GitHub.
2. لا تستخدم أمر بناء.
3. اجعل مجلد النشر هو جذر المشروع.
4. كل تحديث للمستودع يُنشر تلقائيًا.

## قاعدة مهمة لرمز QR

اجعل رمز QR يشير إلى **رابط الموقع الثابت** فقط. لا تجعله يشير مباشرةً إلى رقم هاتف أو حساب اجتماعي؛ عند تحديث محتوى الموقع سيبقى رمز QR صالحًا.
